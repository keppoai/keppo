## Prerequisites

- Node.js 22
- `pnpm`
- Convex CLI
- Docker when validating the local sandbox deployment path
- env management via `dotenvx`

## Local dev flow

```bash
cp .env.example .env.local
pnpm install
pnpm run dev
```

This boots local Convex and the TanStack Start app on port `3000`.

## Required local checks

Before pushing meaningful changes, run:

```bash
pnpm run typecheck
pnpm run check:security
pnpm run check:sdk-type-compat
```
