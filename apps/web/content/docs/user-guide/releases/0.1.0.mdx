## Highlights

- Public `/docs` now ships inside the main web app with built-in search.
- The web runtime is unified under TanStack Start and Vite.
- Approval-first automation flows are documented for operators and contributors.
- Self-hosted guidance now has a dedicated audience section instead of being scattered across repo docs.

## Why this release matters

Before this release, most Keppo documentation was optimized for repo readers. The new docs surface makes product, deployment, and contributor guidance reachable from the public site and dashboard help surfaces without forcing users into internal markdown trees.
